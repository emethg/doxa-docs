title: "¿Qué es el agente?"
description: "Automatiza las actualizaciones de documentación con el agente de Doxa. Activa cambios desde mensajes de Slack, pull requests o llamadas de API."
keywords: ["automation", "automate", "AI", "autoupdate", "maintenance"]

El agente es una herramienta de IA que crea solicitudes de extracción (pull requests) con cambios propuestos para tu documentación a partir de tus indicaciones. Cuando envías una solicitud, el agente:

* **Investiga**: Busca y lee tu documentación existente, los repositorios conectados, el contexto relevante y la web.
* **Planifica**: Crea una lista de tareas estructurada para trabajos de documentación complejos.
* **Escribe**: Genera o actualiza contenido siguiendo los estándares y las mejores prácticas de redacción.
* **Valida**: Ejecuta comprobaciones con Doxa CLI para asegurarse de que la documentación se compile correctamente.
* **Crea una PR**: Abre una solicitud de extracción con cambios propuestos para revisión.

Todos los cambios pasan por solicitudes de extracción (pull requests). El agente nunca realiza confirmaciones directamente en tu branch principal.

<div id="what-you-can-do-with-the-agent">
  ## Qué puedes hacer con el agente
</div>

Usa el agente para:

* Escribir contenido nuevo basado en tus prompts, solicitudes de extracción, hilos de Slack o archivos adjuntos.
* Actualizar documentación existente con nuevas funcionalidades o cambios en la API.
* Procesar e incluir imágenes, diagramas y otros archivos a partir de archivos adjuntos de Slack.
* Buscar y revisar ejemplos de código y referencias de API en toda tu documentación.
* Hacer referencia al código fuente de cualquier repositorio que tenga instalada la aplicación de GitHub de Doxa.
* Obtener contexto en vivo desde aplicaciones conectadas como Slack, Notion, Linear y Jira.
* Automatizar el mantenimiento de la documentación mediante flujos de trabajo.
* Responder preguntas sobre tu documentación y temas de redacción técnica.
* Atender comentarios de revisión de código para mantener la calidad de la documentación.
* Capturar conocimiento de conversaciones en Slack y solicitudes de extracción antes de que se pierda.
* Navegar y coordinar cambios de forma eficiente en múltiples archivos y directorios.

<div id="connect-your-github-account">
  ## Conecta tu cuenta de GitHub
</div>

De forma predeterminada, el agente abre solicitudes de extracción a nombre del bot de Doxa. Para que las solicitudes de extracción se atribuyan a ti, conecta tu cuenta de GitHub en la página [Mi perfil](https://dashboard.doxa.com/settings/account) del dashboard.

<div id="connect-repositories-as-context">
  ## Conecta repositorios como contexto
</div>

El agente solo puede acceder a los repositorios que conectes mediante la Aplicación de GitHub de Doxa. Configura a qué repositorios puede acceder el agente en la sección **Settings** del panel del agente o en la [configuración de la Aplicación de GitHub](https://github.com/apps/doxa/installations/new).

<div id="use-ai-tools-alongside-the-agent">
  ## Usa herramientas de IA junto con el agente
</div>

El agente funciona de forma asíncrona mediante pull requests, pero también puedes usar herramientas de codificación con IA como Cursor o Claude Code localmente para realizar ediciones rápidas e iterativas. Instala el [skill](/es/ai/skillmd) de Doxa y conecta el [servidor MCP](/es/ai/model-context-protocol) para que tu editor tenga el mismo contexto que utiliza el agente.

Para permitir que un agente externo edite tu contenido directamente mediante llamadas a herramientas MCP, conéctalo al [servidor MCP de Doxa](/ai/doxa-mcp).

<Prompt description="Instala el skill de Doxa y los servidores MCP para herramientas de codificación con IA." actions={["copy", "cursor"]}>
Instala el skill de Doxa para obtener contexto sobre la estructura de proyectos, los componentes y las mejores prácticas de documentación de Doxa:

npx skills add https://doxa.com/docs

Luego añade el servidor MCP de Doxa Docs para acceder a la búsqueda en la documentación. https://doxa.com/docs/mcp

Luego añade el servidor MCP de Doxa para acceder al dashboard y a otras herramientas específicas de Doxa. https://mcp.doxa.com
</Prompt>
